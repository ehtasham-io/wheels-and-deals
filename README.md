## 🧭 What is Wheels & Deals?

A fully responsive car dealership website that lets customers browse new and used
cars, explore services, book vehicles, sell their cars, and register for an account —
all without a backend. Built using only HTML5 and CSS3.

---

## ✨ Key Features

- **CSS-only auto-scrolling image slider** — @keyframes animation, pause on hover, no JS
- **Sticky navigation header** — stays visible on scroll across all 10 pages
- **Dropdown menus** — hover-triggered, pure CSS, no JavaScript
- **Brand filter + price sort** — on New and Used car inventory pages
- **30+ car listings** — Honda, Toyota, Mercedes, Suzuki with images and pricing
- **9 spare parts** — Engine, Brakes, Tyres, Headlights, Batteries, Transmission, etc.
- **Google Maps embed** — real UET Narowal location on Contact page
- **3-breakpoint responsive design** — 768px, 480px, 360px
- **Consistent design system** — red (#b30000) brand color, Font Awesome icons, Roboto font
- **CSS Grid + Flexbox** — used appropriately across different page layouts
